Responsive Typography

Create a basic HTML template. Design a heading and a paragraph where the font size of the text adjusts to the width of the viewport using CSS media queries. For instance, the heading text should be larger on a wide screen (above 1024px) and smaller on a narrow screen (below 600px). Ensure the text scales appropriately for a seamless user experience. Use breakpoints like 600px, 768px, and 1024px to define your styles.
