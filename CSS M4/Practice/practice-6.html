Grid Layout for Cards

Design a simple layout that displays six 'card' elements using the CSS Grid Layout. Each card should have a title and a description. Ensure that the cards are displayed in a 3-column grid on large screens (above 1024px), a 2-column grid on medium screens (768px - 1024px), and a single column on small screens (below 768px). Focus on using grid properties such as grid-template-columns and gap.
